6. Studio API Reference | Studio::EventInstance

An instance of an FMOD Studio Event.

Playback Control:

  • Studio::EventInstance::start Starts playback.
  • Studio::EventInstance::stop Stops playback.
  • Studio::EventInstance::getPlaybackState Retrieves the playback state.
  • Studio::EventInstance::setPaused Sets the pause state.
  • Studio::EventInstance::getPaused Retrieves the pause state.
  • Studio::EventInstance::triggerCue Triggers a cue on the event instance.
  • FMOD_STUDIO_STOP_MODE Stop modes.

Playback Properties:

  • Studio::EventInstance::setPitch Sets the pitch multiplier.
  • Studio::EventInstance::getPitch Retrieves the pitch multiplier.
  • Studio::EventInstance::setProperty Sets the value of a built-in property.
  • Studio::EventInstance::getProperty Retrieves the value of a built-in property.
  • Studio::EventInstance::setTimelinePosition Sets the timeline cursor position.
  • Studio::EventInstance::getTimelinePosition Retrieves the timeline cursor position.
  • Studio::EventInstance::setVolume Sets the volume level.
  • Studio::EventInstance::getVolume Retrieves the volume level.
  • Studio::EventInstance::isVirtual Retrieves the virtualization state.
  • FMOD_STUDIO_EVENT_PROPERTY These definitions describe built-in event properties.

3D Attributes:

  • Studio::EventInstance::set3DAttributes Sets the 3D attributes.
  • Studio::EventInstance::get3DAttributes Retrieves the 3D attributes.
  • Studio::EventInstance::setListenerMask Sets the listener mask.
  • Studio::EventInstance::getListenerMask Retrieves the listener mask.

Parameters:

  • Studio::EventInstance::setParameterByName Sets a parameter value by name.
  • Studio::EventInstance::getParameterByName Retrieves a parameter value by name.
  • Studio::EventInstance::setParameterByID Sets a parameter value by unique identifier.
  • Studio::EventInstance::getParameterByID Retrieves a parameter value by unique identifier.
  • Studio::EventInstance::setParametersByIDs Sets multiple parameter values by unique identifier.

Core:

  • Studio::EventInstance::getChannelGroup Retrieves the core channel group.
  • Studio::EventInstance::setReverbLevel Sets the core reverb send level.
  • Studio::EventInstance::getReverbLevel Retrieves the core reverb send level.

Profiling:

  • Studio::EventInstance::getCPUUsage Retrieves the event CPU usage data.
  • Studio::EventInstance::getMemoryUsage Retrieves memory usage statistics.

Callbacks:

  • Studio::EventInstance::setCallback Sets the user callback.
  • Studio::EventInstance::setUserData Sets the event instance user data.
  • Studio::EventInstance::getUserData Retrieves the event instance user data.
  • FMOD_STUDIO_EVENT_CALLBACK Callback that is fired when a Studio::EventInstance changes state.
  • FMOD_STUDIO_PLUGIN_INSTANCE_PROPERTIES Describes a DSP plugin instance.
  • FMOD_STUDIO_PROGRAMMER_SOUND_PROPERTIES Describes a programmer sound.
  • FMOD_STUDIO_TIMELINE_BEAT_PROPERTIES Describes a beat on the timeline.
  • FMOD_STUDIO_TIMELINE_MARKER_PROPERTIES Describes a marker on the timeline.
  • FMOD_STUDIO_EVENT_CALLBACK_TYPE Studio event callback types.

General:

  • Studio::EventInstance::getDescription Retrieves the event description.
  • Studio::EventInstance::release Marks the event instance for release.
  • Studio::EventInstance::isValid Checks that the EventInstance reference is valid.

  unsigned int mask
);

RESULT Studio.EventInstance.setListenerMask(
  uint mask
);

Studio.EventInstance.setListenerMask(
  mask
);
mask

Listener mask.

  • Default: 0xFFFFFFFF

The listener mask controls which listeners are considered when calculating 3D panning and the values of listener relative automatic parameters.

To create the mask you must perform bitwise OR and shift operations, the basic form is 1 << listener_index ORd together with other required listener indices.
For example to create a mask for listener index 0 and 2 the calculation would be mask = (1 << 0) | (1 << 2), to include all listeners use the default mask of 0xFFFFFFFF.

See Also: Studio::EventInstance::getListenerMask
